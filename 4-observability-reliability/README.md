## 🛠️ Key Observability Skills
* **Monitoring Dashboards**: Creating Grafana visuals representing CPU load, memory limits, network traffic, and HTTP response distributions.
* **Alert Orchestration**: Writing Alertmanager rules firing on high latency spikes or memory exhaustion thresholds.
* **Log Consolidation**: Deploying Fluentd agents forwarding system traces to Elasticsearch nodes.
* **APM Diagnostics**: Tracing database queries using OpenTelemetry hooks.
