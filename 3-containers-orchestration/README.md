## 🛠️ Key Orchestration Skills
* **Container Optimization**: Writing thin Docker images using multi-stage compile structures and non-root execution environments.
* **Kubernetes Control**: Managing deployments, declaring service discovery resources, and configuring Ingress controllers.
* **CI/CD Pipeline Design**: Designing automated triggers executing linting checks, unit tests suites, building images, and pushing to registries.
* **Declarative Synchronization**: Applying GitOps principles to align active Kubernetes cluster structures with Git states using ArgoCD.
