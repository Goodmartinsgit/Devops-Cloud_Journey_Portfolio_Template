- [ ] **Task 3: Pass AWS Cloud Practitioner**
  * **Objective**: **What this task is:** The AWS Cloud Practitioner (CLF-C02) is the entry-level AWS certification. It tests foundational cloud knowledge — what cloud computing is, what the major AWS service categories are, and the basics of AWS pricing and support.
  * **Status**: ⏳ Planned
  * **Task Folder**: `[task-03-pass-aws-cloud-practitioner/](./tasks/task-03-pass-aws-cloud-practitioner/)`
- [ ] **Task 4: Pass AWS Solutions Architect Associate (Target: 85%+ on Practice Exams Before Booking)**
  * **Objective**: **What this task is:** The most important AWS certification for a Cloud/DevOps engineer. It signals that you understand how to design cloud systems — not just that you know what services exist, but why and how to use them together.
  * **Status**: ⏳ Planned
  * **Task Folder**: `[task-04-pass-aws-solutions-architect-associate-target/](./tasks/task-04-pass-aws-solutions-architect-associate-target/)`
- [ ] **Task 5: Pass HashiCorp Terraform Associate**
  * **Objective**: **What this task is:** The Terraform Associate certifies your understanding of Infrastructure as Code principles and Terraform's workflow, language, and state management.
  * **Status**: ⏳ Planned
  * **Task Folder**: `[task-05-pass-hashicorp-terraform-associate/](./tasks/task-05-pass-hashicorp-terraform-associate/)`
- [ ] **Task 6: Pass CNCF CKA — Use killer.sh for 2 Weeks of Daily Practice**
  * **Objective**: **What this task is:** The Certified Kubernetes Administrator tests whether you can administer a Kubernetes cluster — installing it, configuring it, troubleshooting it, and keeping it running reliably.
  * **Status**: ⏳ Planned
  * **Task Folder**: `[task-06-pass-cncf-cka-use-killersh-for/](./tasks/task-06-pass-cncf-cka-use-killersh-for/)`
- [ ] **Task 7: Pass CNCF CKAD**
  * **Objective**: **What this task is:** The Certified Kubernetes Application Developer tests whether you can deploy, configure, and manage applications on Kubernetes — from a developer or application team perspective.
  * **Status**: ⏳ Planned
  * **Task Folder**: `[task-07-pass-cncf-ckad/](./tasks/task-07-pass-cncf-ckad/)`
- [ ] **Task 8: Pass GitHub Actions Certification**
