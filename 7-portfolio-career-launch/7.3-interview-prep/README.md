# Interview Prep

This subdirectory houses SRE system design answers, programming challenges solutions, and interview post-mortem cheat sheets.

## 📋 Core Prep Categories
* **Linux Internals**: Process isolation, file systems parameters, diagnostic flags, system call tracking (`strace`).
* **Networking Scenarios**: Root cause checks for HTTP failure codes, proxy overrides, MTU configurations, and network delays.
* **System Design Case Studies**: Architecture write-ups for high-traffic servers, caching configurations, event processing.
* **Coding Practice**: Python and Bash solutions for standard text-processing and systems automation problems.
