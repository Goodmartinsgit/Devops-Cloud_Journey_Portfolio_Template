## 🛠️ Key Architectural Skills
* **Event Streams**: Setting up Apache Kafka clusters to buffer high-throughput message events across backend microservices.
* **Database Replications**: Designing master-slave replication scripts for Postgres database pools with automatic failovers.
* **Cost Optimizations**: Writing automation scripts deleting unused volumes and auditing cloud costs relative to budget limits.
* **Failover Auditing**: Documenting multi-region disaster recovery routes ensuring RTO/RPO targets.
