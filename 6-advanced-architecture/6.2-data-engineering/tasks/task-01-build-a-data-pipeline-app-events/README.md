# Task 1: Build a Data Pipeline (App Events → Kafka → Kinesis Firehose → S3 → Athena → Grafana)

## Objective

Build an end-to-end data pipeline that ingests application events, streams them through Kafka and Kinesis Firehose, lands them in S3, makes them queryable via Athena, and visualises them in Grafana.
