## 🛠️ Key Cloud Engineering Skills
* **Infrastructure Design**: Provisioning secure, multi-tier virtual private networks (VPCs) featuring private/public subnet isolation and NAT configurations.
* **Identity Management (IAM)**: Implementing least-privilege policies, delegating trust permissions via assume roles, and tracking API activity.
* **Infrastructure as Code (IaC)**: Automating consistent multi-resource clouds configurations using Terraform state management.
* **Configuration Management (CM)**: Writing idempotence-preserving Ansible Playbooks to configure remote web stacks and system components.
